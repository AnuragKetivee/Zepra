# ZepraBrowser Project Documentation

Welcome to the central documentation directory for **ZepraBrowser**, **ZebraScript**, **NXRender**, and the broader **NeolyxOS** ecosystem.

## 🧭 Developer Hub

The best way to navigate this documentation is by opening the [`index.html`](index.html) Developer Hub using a Live Server. The Hub provides a modern, interactive, and searchable interface to all architectural documentation, APIs, and guidelines.

## 🔑 Key Subsystems

- **[ZepraScript Engine Documentation](zeprascript/index.html)**
  High-performance JS runtime with multi-tier JIT, Temporal API, and full WASM 2.0 support.
- **[System Architecture](Architecture.html)**
  Multi-process IPC layout, capability sandboxing, site-isolation, frame routing, and process definitions.
- **[NXRender Pipeline](NXRENDER/ARCHITECTURE.html)**
  Hardware-accelerated rendering pipeline for NeolyxOS with GPU compositing and damage tracking.
- **[Developer Tools & Admin](ADVANCED_DEVELOPER_TOOLS.html)**
  Native debugging interfaces, remote inspection protocol, and performance profiling workflows.
