Overview

ZepraScript's AI subsystem bridges the gap between traditional scripting and generative AI. Instead of relying on external cloud APIs with high latency and costs, ZepraScript is designed to work seamlessly with Ollama running locally on your machine.

This allows you to verify code, summarize text, generate content, and create intelligent agents directly within your ZepraScript applications, with zero data leaving your device.

Configuration

Before using the AI APIs, you need to point ZepraScript to your Ollama instance. By default, it looks for Ollama on localhost:11434.

import { ai } from 'zepra:ai';

// Configure the provider (optional if using defaults)
ai.configure({
    provider: 'ollama',
    endpoint: 'http://localhost:11434',
    defaultModel: 'llama3' // or 'mistral', 'gemma', etc.
});

Ollama Integration

Ensure you have Ollama installed and a model pulled. Run this in your terminal: 

ollama run llama3

Once the model is ready, ZepraScript can communicate with it immediately.

API Reference

ai.chat(options)

Promise<ChatResponse> chat(options: ChatOptions)

Sends a chat prompt to the model and returns the response.

Parameters

  • model (string, optional): The model to use (default: configured default).
  • messages (Array): Array of message objects { role: 'user'|'assistant'|'system', content: string }.
  • stream (boolean): Whether to stream the response (default: false).

ai.generate(prompt)

Promise<string> generate(prompt: string, options?: GenOptions)

Simple completion method for single-turn tasks.

ai.embeddings(input)

Promise<number[]> embeddings(input: string)

Generates vector embeddings for the given text, useful for semantic search and RAG applications.

Real-World Examples

1. Intelligent Text Summarizer

Summarize long articles or logs using Llama 3.

import { ai } from 'zepra:ai';

async function summarizeLog(logContent) {
    console.log("Analyzing logs...");
    
    const summary = await ai.chat({
        model: 'llama3',
        messages: [
            { role: 'system', content: 'You are a devops assistant. Summarize the following error log concisely.' },
            { role: 'user', content: logContent }
        ]
    });

    return summary.message.content;
}

// Usage
const errorLog = "Error: Connection timed out at 14:00. Retrying... Failed. Database unreachable.";
summarizeLog(errorLog).then(console.log);

2. Code Review Agent

Ask the AI to review a snippet of code.

const codeSnippet = `
function add(a, b) {
    return a + b;
}`;

const review = await ai.generate(
    `Review this JavaScript code and suggest improvements:\n\n${codeSnippet}`
);

console.log(review);

3. Semantic Search (RAG)

Using embeddings to find related documents.

const docs = ["Login page logic", "Database schema", "User authentication flow"];
const query = "How do users sign in?";

// Generate embeddings
const queryVec = await ai.embeddings(query);
const docVecs = await Promise.all(docs.map(d => ai.embeddings(d)));

// Calculate similarity (pseudo-code for cosine similarity)
const bestMatch = findClosestVector(queryVec, docVecs);
console.log("Most relevant doc:", docs[bestMatch.index]);
